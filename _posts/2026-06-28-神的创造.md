<span style="color:#C80713; font-size: 23px; font-weight: bold;">一、天地万物的被造</span>

<span style="color:#C80713; font-size: 21px; font-weight: bold;">1、错误的宇宙形成观</span>

关于宇宙的起源，世上有各种各样的说法，但归纳起来主要有以下几种错误观点：

**进化论**：认为宇宙万物是从无到有、从低级到高级逐渐进化而来的。但进化论无法解释生命的起源、宇宙的精妙设计以及生物大爆炸等根本问题。

**大爆炸理论**：认为宇宙起源于一次大爆炸，但大爆炸的能量从哪里来？爆炸的精确度为何如此之高？这些都无法解释。

**轮回说**：认为一切都是循环的，没有开始也没有结束。但这与热力学第二定律（熵增定律）相矛盾。

**宿命论/偶然论**：认为一切都是偶然发生的，或者一切都是命中注定的。但偶然无法产生有序的宇宙，宿命论则否定了人的自由意志。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">2、宇宙被造的过程</span>

圣经创世记第一章清楚地告诉我们宇宙被造的过程：

<span style="color:#2307D2;">起初神创造天地。（创1:1）</span>

这是圣经的第一句话，也是整个圣经的根基。它告诉我们：宇宙有开始——起初；宇宙有创造者——神；宇宙是被造的——创造。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">第一日：造光</span>

<span style="color:#2307D2;">神说，要有光，就有了光。神看光是好的，就把光暗分开了。神称光为昼，称暗为夜。有晚上，有早晨，这是头一日。（创1:3-5）</span>

光是神创造的第一样东西。这里我们看到神创造的方式——"神说"。神用话语创造，这告诉我们神的能力在祂的话语中。希伯来书11:3说："我们因着信，就知道诸世界是借神话造成的。"

<span style="color:#C80713; font-size: 21px; font-weight: bold;">第二日：造空气</span>

<span style="color:#2307D2;">神说，诸水之间要有空气，将水分为上下。神就造出空气，将空气以下的水、空气以上的水分开了。事就这样成了。（创1:6-7）</span>

第二日神创造了空气（穹苍），把水分开。空气是维持生命所必需的，没有空气，一切生命都无法存活。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">第三日：分旱地与造植物</span>

<span style="color:#2307D2;">神说，天下的水要聚在一处，使旱地露出来。事就这样成了。神称旱地为地，称水的聚处为海。神看着是好的。神说，地要发生青草，和结种子的菜蔬，并结果子的树木，各从其类，果子都包着核。事就这样成了。于是地发生了青草，和结种子的菜蔬，各从其类，并结果子的树木，各从其类，果子都包着核。神看着是好的。有晚上，有早晨，是第三日。（创1:9-13）</span>

第三日做了两件事：分旱地、造植物。注意这里的顺序——先造环境（旱地），再造需要环境的生物（植物）。神做事有先后的次序，祂先为人预备好一切所需要的环境和条件，才造人。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">第四日：造光体</span>

<span style="color:#2307D2;">神说，天上要有光体，可以分昼夜，作记号，定节令，日子，年岁。并要发光在天空，普照在地上。事就这样成了。于是神造了两个大光，大的管昼，小的管夜。又造众星。就把这些光摆列在天空，普照在地上，管理昼夜，分别明暗。神看着是好的。有晚上，有早晨，是第四日。（创1:14-19）</span>

第四日神创造了太阳、月亮和众星。有趣的是，光是在第一日造的，而光体（太阳、月亮）是在第四日才造的。这说明光的来源不是太阳，而是神。神本身就是光，太阳只是神所造的发光体之一。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">第五日：造水中生物与空中飞鸟</span>

<span style="color:#2307D2;">神说，水要多多滋生有生命的物，要有雀鸟飞在地面以上，天空之中。神就造出大鱼，和水中所滋生各样有生命的动物，各从其类。又造出各样飞鸟，各从其类。神看着是好的。神就赐福给这一切，说，滋生繁多，充满海中的水。雀鸟也要多生在地上。有晚上，有早晨，是第五日。（创1:20-23）</span>

第五日神创造了水中生物和空中飞鸟，并赐福给它们，让它们滋生繁多。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">第六日：造陆地动物与人</span>

<span style="color:#2307D2;">神说，地要生出活物来，各从其类。牲畜，昆虫，野兽，各从其类。事就这样成了。于是神造出野兽，各从其类。牲畜，各从其类。地上一切昆虫，各从其类。神看着是好的。（创1:24-25）</span>

<span style="color:#C80713; font-size: 21px; font-weight: bold;">第七日：安息</span>

<span style="color:#2307D2;">天地万物都造齐了。到第七日，神造物的工已经完毕，就在第七日歇了他一切的工，安息了。神赐福给第七日，定为圣日，因为在这日神歇了他一切创造的工，就安息了。（创2:1-3）</span>

第七日神安息了。这不是说神累了需要休息，而是神完成了创造的工作，进入了安息。安息日是神赐给人的一个礼物，让人也有安息。

<span style="color:#C80713; font-size: 23px; font-weight: bold;">二、男人的被造</span>

<span style="color:#2307D2;">神说，我们要照着我们的形像，按着我们的样式造人，使他们管理海里的鱼，空中的鸟，和地上各样行动的活物。神就照着自己的形像造人，乃是照着他的形像造男造女。神就赐福给他们，又对他们说，要生养众多，遍满地面，治理这地。也要治理海里的鱼，空中的鸟，和地上各样行动的活物。（创1:26-28）</span>

关于人的被造，圣经给了我们两个版本：创世记第一章是概括性的记述，第二章是详细的记述。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">1、神造人的目的</span>

神造人有两个目的：

**荣耀神**：以赛亚书43:7说："凡称为我名下的人，是我为自己的荣耀创造的。"人被造最大的目的就是荣耀神。

**与神相交**：神造人是为了与人建立关系，神是个灵，祂造了有灵的人，就是为了与人相通、与人相交。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">2、神造人的方式</span>

<span style="color:#2307D2;">耶和华神用地上的尘土造人，将生气吹在他鼻孔里，他就成了有灵的活人。（创2:7）</span>

神造人的方式非常特别，分两步：

**用地上的尘土造人的身体**：人身体最原始的材料就是地上的尘土，这告诉我们人的身体是出于地，也要归于地。

**将生气吹入鼻孔**：这"生气"就是神的灵，神将祂的灵吹入人的鼻孔，人就成了"有灵的活人"。这一步让人与其他所有受造物区别开来——人有灵，可以与神相通。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">3、神造人的特殊性</span>

人不同于其他受造物，人有两个特殊性：

**人有神的形像和样式**：这是人区别于其他一切受造物的最大特征。神的形像包括：
- 理性：人有思考、推理、判断的能力
- 道德：人有是非对错的观念
- 情感：人有爱、恨、喜、怒等情感
- 意志：人有选择和决定的能力
- 创造性：人能创造、发明、艺术等

**人被赋予管理权**：神让人管理海里的鱼、空中的鸟、地上的走兽。人被造不是为管理神，而是为管理神所造的万物。这告诉我们人在受造界中有尊贵的地位。

<span style="color:#C80713; font-size: 23px; font-weight: bold;">三、女人的被造</span>

<span style="color:#2307D2;">耶和华神说，那人独居不好，我要为他造一个配偶帮助他。（创2:18）</span>

<span style="color:#C80713; font-size: 21px; font-weight: bold;">1、女人被造的原因</span>

神说"那人独居不好"，这是圣经第一次出现"不好"这个词。在前五日的创造中，神每次创造完都说"神看着是好的"，但到了第六日，神说"那人独居不好"。这说明：

**人需要关系**：神造人是有社交需求的，人不是被造来独居的。

**婚姻是神设立的**：神亲自为人设立婚姻，婚姻不是人的发明，而是神的安排。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">2、女人被造的过程</span>

<span style="color:#2307D2;">耶和华神使他沉睡，他就睡了。于是取下他的一条肋骨，又把肉合起来。耶和华神就用那人身上所取的肋骨，造成一个女人，领她到那人跟前。（创2:21-22）</span>

女人被造的过程也很特别：

**神让亚当沉睡**：这"沉睡"在原文中有"被神降下深度睡眠"的意思，可能带有一种牺牲的预表——基督为我们舍己，正如亚当为夏娃"舍己"（肋骨被取走）。

**用肋骨造成**：肋骨是靠近心脏的地方，这说明女人是男人最亲近、最珍贵的伴侣。肋骨也起到保护内脏的作用，说明女人对男人有保护、支持的作用。

<span style="color:#C80713; font-size: 21px; font-weight: bold;">3、男女的关系</span>

<span style="color:#2307D2;">因此，人要离开父母，与妻子连合，二人成为一体。（创2:24）</span>

这节经文告诉我们婚姻的三个原则：

**离开**：人要离开父母，建立独立的家庭单位。

**连合**：夫妻要紧密联合，彼此忠诚，不可分割。

**成为一体**：这不仅指身体的结合，更指心灵、情感、意志的全面合一。

<span style="color:#C80713;">神造男造女，男女平等又有不同的角色和功能。男人是头，女人是帮助者；男人要爱妻子，女人要敬重丈夫。这是神设立的正常秩序，当这个秩序被颠倒时，家庭和社会就会出现混乱。</span>
