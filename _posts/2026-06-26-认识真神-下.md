#### <span style="font-size: 23px; color: #DC1717;"><strong>三、神的属性</strong></span>

<span style="color: #000000;">神的属性分为**<span style="color: #DC1717;">神本体的属性</span>**和**<span style="color: #DC1717;">神灵在的属性</span>**。</span>

#### <span style="font-size: 21px; color: #DC1717;"><strong>神本体的属性</strong></span>

<span style="color: #000000;">神本体的属性又叫神超越的属性，包括神的无限全能、自有永有、无所不知、无所不在、永恒不变、独一无二。这些神本体的属性是神独有的，是人不能分享和效法的。</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>无限全能 无所不能</strong></span>

<span style="color: #450FF5;">亚伯兰年九十九岁的时候，耶和华向他显现，对他说："我是全能的神，你当在我面前作完全人。"（创17:1）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>自有永有</strong></span>

<span style="color: #450FF5;">神对摩西说："我是自有永有的"。（出3:14）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>无所不知 无所不在</strong></span>

<span style="color: #450FF5;">彼得因为耶稣第三次对他说"你爱我吗"，就忧愁，对耶稣说："主啊，你是无所不知的，你知道我爱你。"（约21:17）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>永恒不变</strong></span>

<span style="color: #450FF5;">惟有你永不改变，你的年数没有穷尽。（诗篇102:27）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>独一无二</strong></span>

<span style="color: #450FF5;">以色列啊，你要听！耶和华我们神是独一的主。（申6:4）</span>

#### <span style="font-size: 21px; color: #DC1717;"><strong>神灵在的属性</strong></span>

<span style="color: #000000;">神灵在的属性又称为神道德的属性，包括神的慈爱、信实、公义、正直、圣洁、良善、恩慈、怜悯、慈悲。这些神道德的属性是神要人效法的。不过人无论怎样效法，人都不可能达到与神一致，人活出来的这些神美好的性情都是相对的，只有神是绝对的。如，神的公义是绝对完全的公义，人的公义是相对的。</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>慈爱</strong></span>

<span style="color: #450FF5;">应当称谢耶和华，因他本为善，他的慈爱永远长存。（代上16:34）</span>    
<span style="color: #450FF5;">你看父赐给我们是何等的慈爱，使我们得称为神的儿女；我们也真是他的儿女。世人所以不认识我们，是因未曾认识他。（约一3:1）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>信实</strong></span>

<span style="color: #450FF5;">所以你要知道耶和华你的神，他是神，是信实的神，向爱他、守他诫命的人守约，施慈爱直到千代；（申7:9）</span>  
<span style="color: #450FF5;">神是信实的，你们原是被他所召，好与他儿子我们的主耶稣基督一同得份。（林前1:9）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>公义</strong></span>

<span style="color: #450FF5;">他是磐石，他的作为完全，他所行的无不公平，是诚实无伪的神，又公义，又正直。（申32:4）</span>  
<span style="color: #450FF5;">公义的父啊，世人未曾认识你，我却认识你，这些人也知道你差了我来。（约17:25）</span>  
<span style="color: #450FF5;">神既是公义的，就必将患难报应那加患难给你们的人；（帖后1:6）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>正直</strong></span>

<span style="color: #450FF5;">耶和华是良善正直的，所以他必指示罪人走正路。（诗篇25:8）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>圣洁</strong></span>

<span style="color: #450FF5;">你晓谕以色列全会众说：你们要圣洁，因为我耶和华你们的神是圣洁的。（利19:12）</span>  
<span style="color: #450FF5;">你们弃绝了那圣洁公义者，反求着释放一个凶手给你们。（徒3:14）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>良善</strong></span>

<span style="color: #450FF5;">耶和华是良善正直的，所以他必指示罪人走正路。（诗篇25:8）</span>  
<span style="color: #450FF5;">耶稣对他说：除了神一位之外，再没有良善的。（可10:18）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>恩慈</strong></span>

<span style="color: #450FF5;">耶和华说："我要显我一切的恩慈，在你面前经过，宣告我的名。我要恩待谁，就恩待谁；要怜悯谁，就怜悯谁。"（出33:19）</span>  
<span style="color: #450FF5;">要将他极丰富的恩典，就是他在基督耶稣里向我们所施的恩慈，显明给后来的世代看。（弗2:7）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>怜悯</strong></span>

<span style="color: #450FF5;">耶和华在他面前宣告说："耶和华，耶和华，是有怜悯、有恩典的神，不轻易发怒，并有丰盛的慈爱和诚实。"（出34:6）</span>  
<span style="color: #450FF5;">然而神既有丰富的怜悯，因他爱我们的大爱，（弗2:4）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>慈悲</strong></span>

<span style="color: #450FF5;">耶和华善待万民，他的慈悲覆庇他一切所造的。（诗篇145:9）</span>  
<span style="color: #450FF5;">愿颂赞归与我们的主耶稣基督的父神，就是发慈悲的父，赐各样安慰的神。（林后1:3）</span>

#### <span style="font-size: 21px; color: #DC1717;"><strong>耶稣基督</strong></span>

<span style="color: #000000;">耶稣基督除了有神本体的属性和灵在的属性外，还有一个独特的属性——神人二性，祂是完全的神，也是完全的人。</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>耶稣基督的七个"我是"</strong></span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>1. 我是生命的粮</strong></span>

<span style="color: #450FF5;">我就是生命的粮。（约6:48）</span>  
<span style="color: #450FF5;">我是从天上降下来生命的粮；人若吃这粮，就必永远活着。我所要赐的粮，就是我的肉，为世人之生命所赐的。（约6:51）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>2. 我是世界的光</strong></span>

<span style="color: #450FF5;">耶稣又对众人说："我是世界的光。跟从我的，就不在黑暗里走，必要得着生命的光。"（约8:12）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>3. 我是羊的门</strong></span>

<span style="color: #450FF5;">所以耶稣又对他们说："我实实在在地告诉你们：我就是羊的门。"</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>4. 我是好牧人</strong></span>

<span style="color: #450FF5;">我是好牧人，好牧人为羊舍命。（约10:11）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>5. 我是复活、我是生命</strong></span>

<span style="color: #450FF5;">耶稣对她说："复活在我，生命也在我；信我的人，虽然死了，也必复活。"（约11:25）</span>

#### <span style="font-size: 22px; color: #DC1717;"><strong>6. 我是道路、真理、生命</strong></span>

<span style="color: #450FF5;">耶稣说："我就是道路、真理、生命；若不藉着我，没有人能到父那里去。"（约14:6）</span>

#### <span style="font-size: 20px; color: #DC1717;"><strong>7. 我是真葡萄树</strong></span>

<span style="color: #450FF5;">我是真葡萄树，我父是栽培的人。（约15:1）</span>  
<span style="color: #450FF5;">我是葡萄树，你们是枝子；常在我里面的，我也常在他里面，这人就多结果子；因为离了我，你们就不能作什么。（约15:5）</span>

#### <span style="font-size: 23px; color: #DC1717;"><strong>四、神的工作</strong></span>

<span style="color: #000000;">神的工作包括神的创造、救赎、审判和护理。关于神的工作，特别是神的创造和救赎，我将在以后的基要真理里逐一详细分享。</span>
